# Senior Developer & Senior UI/UX Designer Mode

I am not looking for another AI-generated agency website.

The current website feels like it was generated by AI.

I want you to redesign and refine the website as if it were built by:

* A senior frontend engineer with 10+ years of experience
* A senior UI/UX designer from a top digital agency
* An Awwwards-winning web designer

Do not think like an AI website builder.

Think like an experienced professional creating a website for a real paying client.

---

## Most Important Requirement

Avoid all common AI-generated website patterns.

Avoid:

* Generic cards
* Repetitive sections
* Equal spacing everywhere
* Template-like layouts
* Basic gradients
* Predictable designs
* Generic icons
* Standard hero sections
* Stock AI aesthetics

The website should feel handcrafted.

---

## Visual Goal

When someone visits WebAksh they should think:

"An experienced design agency built this."

Not:

"This looks AI generated."

---

## Design References

Study and take inspiration from:

* Linear
* Stripe
* Cuberto
* Lusion
* Refokus
* Bruno Simon
* Locomotive
* Awwwards agency websites

Do not copy.

Use these references for quality level, motion design, spacing, and polish.

---

## Improve Layout Composition

Every section should have unique structure.

Avoid:

* Section title
* Three cards
* Button

Repeated everywhere.

Create visual variety.

Use:

* Asymmetrical layouts
* Layered content
* Alternating compositions
* Editorial style sections
* Creative image placements

---

## Improve Typography

Typography should feel premium.

Use:

* Large display headlines
* Strong hierarchy
* Better line heights
* Better letter spacing
* More breathing room

Typography should carry the design.

---

## Improve Motion Design

Animations should feel intentional.

Add:

* Text reveal animations
* Smooth content transitions
* Scroll storytelling
* Image reveal effects
* Cursor interactions
* Depth and layering

Avoid random animations.

Every animation must support the experience.

---

## Improve Project Portfolio

Projects should feel like real case studies.

Not simple cards.

When opening a project:

Show:

* Business problem
* Design process
* Technology stack
* User experience decisions
* Challenges solved
* Results achieved

Make visitors feel confident hiring WebAksh.

The website should feel like a real agency.

Code quality must feel production-ready.

Requirements:

* Clean component architecture
* Reusable design system
* Proper animations
* Responsive layouts
* Accessibility

No shortcuts.

Build as if this project will be reviewed by senior frontend engineers.

---

## Final Goal

The website should not look AI generated.

It should look like a real senior designer and senior frontend developer spent weeks crafting it.

Every section should feel intentional, premium, polished, and memorable.

Prioritize craftsmanship over quantity.

Quality over features.

Polish over complexity.
